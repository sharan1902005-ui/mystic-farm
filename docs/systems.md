# Game Systems

## Core

- Canvas game loop
- Delta-time FPS counter
- F1 developer panel
- Save slots with automatic backups
- PWA manifest and service worker

## World

- Procedural farm decoration
- Procedural mine generation
- Weather and seasons
- Festivals and dynamic market prices

## Player

- Inventory
- Energy and health
- Tools, equipment, skills, and talents

## Gameplay

- Farming
- Fishing
- Mining
- Combat
- Crafting/cooking
- Animals
- Museum collections
- Achievements
